Data & Privacy

Personal blog privacy statement. Basically, yes I use some tools - no I don't need to annoy you with cookie banners.

Privacy, data ownership, tracking etc are all topics which I find fascinating.

In terms of this blog, it's my understanding I do not need to annoy you with cookie banners and such as this blog does not make money - https://www.nomipalony.com/gdpr-for-bloggers/

I do currently use Google Analytics to see how unpopular my site is, but I've taken steps to not use cookies - https://www.acmconsulting.eu/post/hate-cookie-banners-me-too/

The social buttons are simple pre-filled URLs so do not send any data back to the platforms.

Unsplash (the free images I use) seems to be setting a cookie:

This cookie is set by the provider Unsplash. This cookie is used for enabling the video content on the website.

I will see if I can disable/opt out of that, but so far have not been able to.

Cloudflare (my CDN) are also setting a cookie:

The cookie is used by cdn services like CloudFare to identify individual clients behind a shared IP address and apply security settings on a per-client basis. It does not correspond to any user ID in the web application and does not store any personally identifiable information.

(descriptions of the cookies from the excellent : https://www.cookieserve.com/)

That's it (I think).

If you have a question about the data privacy, usage or otherwise general questions, comments, issues, please follow the instructions of the contact page to drop me a message.
